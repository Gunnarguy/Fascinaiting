This website showcases OpenAssistant, a comprehensive iOS application that brings the full power of OpenAI's Assistants API directly to your iPhone. The website provides detailed information about:

- **Features**: Comprehensive assistant management, vector stores, file processing, and real-time chat
- **Architecture**: Deep dive into the MVVM architecture and SwiftUI implementation
- **Component Interactions**: Interactive visualization of the app's architecture
- **Support**: FAQs, troubleshooting guides, and contact information

## Local Development

To run the website locally:

1. Clone the repository
2. Start a local web server:
   ```bash
   python3 -m http.server 8080
   ```
3. Open your browser to `http://localhost:8080`

Alternatively, you can use any static file server or open the HTML files directly in your browser.

## Design Philosophy

- **Privacy First**: No tracking, no analytics, no third-party data collection
- **Performance**: Optimized for fast loading and smooth animations
- **Accessibility**: Built with semantic HTML and ARIA labels
- **Mobile Responsive**: Beautiful experience on all device sizes
- **Modern Design**: Clean, professional interface following Apple's design guidelines
